# daw-capa-web

## Descripción

Este skill se enfoca en la definición de la capa web para una aplicación web utilizando Spring Boot.

### Cuando usar este skill

- Cuando se necesite definir la capa web de una aplicación web.
- Cuando se quiera estructurar la lógica de presentación de una aplicación utilizando Spring Boot.

## Contexto

La capa web es la parte de una aplicación web que se encarga de manejar las solicitudes HTTP, procesar los datos de entrada, interactuar con la capa de negocio y devolver las respuestas HTTP. En Spring Boot, la capa web se puede implementar utilizando controladores REST, controladores MVC, etc.

## Información adicional

- Usar las skills de controladores y validación para definir los controladores y las validaciones de la capa web.
- La capa web debe ser lo más delgada posible, delegando la lógica de negocio a los servicios o casos de uso, y la lógica de acceso a datos a los DAOs.
